Background

WebKit & Apple

"The WebKit project was started within Apple by Don Melton on June 25, 2001 as a fork of KHTML and KJS. Melton explained in an e-mail to KDE developers that KHTML and KJS allowed easier development than other available technologies by virtue of being small (fewer than 140,000 lines of code), cleanly designed and standards-compliant." - Wikipedia (Webkit)

Building WebKit

1. install dev tools

2. download the source code

$ git clone git://git.webkit.org/WebKit.git WebKit

$ cd webkit

3. build!

$ build-webkit --debug
